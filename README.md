# UNIBE - Algoritmo Recursivo en Python

## Descripción del Proyecto

Este repositorio contiene la implementación de un programa desarrollado en **Python** que utiliza un **algoritmo recursivo**. El objetivo del proyecto es demostrar la comprensión del uso de la recursión en programación mediante la creación de un programa funcional.

Además, la asignación incluye una investigación sobre el concepto de **programación concurrente** y su importancia en el desarrollo poniendo el ejemplo de sucesión de fibonacci
